# scada-platform 项目知识库

## 项目定位

`scada-platform` 面向工业现场或飞机装配场景的可视化监控、设备状态展示、报警、交互控制和现场态势呈现。

核心目标是把复杂现场对象以清晰、稳定、可操作的界面呈现出来，而不是只做静态大屏。
