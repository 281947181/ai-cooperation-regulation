# scada-platform 业务规约

## 核心对象

典型对象包括：

- 飞机部段本体。
- 装配型架平台。
- 制孔机器人。
- 调姿定位器。
- 移载 AGV。
- 工位、产线、区域。

对象建模必须优先服务于状态展示、定位、报警和操作闭环。

## 实时状态

实时状态至少包含：

- 对象标识。
- 状态值。
- 状态时间。
- 质量码或有效性。
- 报警级别。
- 数据来源。

不得假设所有设备数据稳定到达。

## 交互分类

SCADA 界面交互必须明确区分：

- 只读监视。
- 本地操作。
- 远程控制。
- 高风险控制。

高风险操作必须有二次确认、权限校验和操作记录。
