# wealth_advisor 开发规约

## 基本要求

本项目开发必须遵守：

```text
constitution.md
runtime.md
projects/wealth_advisor/README.md
```

## 远程协作

该仓库默认分支为 `main`。每轮任务必须以远程 `main` 最新状态为基线，完成后默认提交并推送远程。

## 开发约束

- 涉及金融、价格、市场、政策等信息时，必须确认数据来源和时间。
- 不得把示例数据伪装成真实数据。
- 不得在未确认安全边界前存储敏感个人信息。
- 不得生成缺少风险提示的投资结论。
- 如涉及外部 API，必须处理限流、失败、重试和数据过期。
