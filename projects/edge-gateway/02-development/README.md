# edge-gateway 开发规约

## 协议适配约束

OPC UA：

- 优先使用成熟 SDK。
- 关注 endpoint、证书、session、subscription 生命周期。
- 避免一点一连接或一设备一线程的低效设计。

Modbus TCP / S7：

- 默认轮询模型。
- 点位合并读取。
- 控制并发和超时。
- 统一错误码和状态上报。

## Codex 约束

- 不得把协议连接逻辑写死在业务 Controller 中。
- 不得为每个点位创建独立线程。
- 不得绕过配置版本管理直接下发临时配置。
- 日志必须包含网关、设备、点位等关键定位信息。
- 网络异常、认证异常、协议异常必须分类处理。
