# edge-gateway 项目知识库

## 项目定位

`edge-gateway` 是边缘网关与中心侧配置平台方向的项目知识库，面向工业现场协议适配、中心侧配置管理、配置下发、数据采集、清洗、存储和上报。

核心目标是建立稳定、可部署、可维护的边缘到中心数据链路。
