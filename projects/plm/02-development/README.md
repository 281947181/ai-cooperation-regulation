# plm 开发规约

## 基本要求

本项目开发必须遵守：

```text
constitution.md
runtime.md
projects/plm/README.md
```

## 当前约束

- 任务开始前必须从远程 main 分支同步最新代码。
- 任务完成后默认必须提交并推送远程仓库。
- 未确认代码结构前，不得大范围重构。
- 不得把产品化能力和客户定制能力混在同一个无边界模块中。
- 如果涉及 Docker，遵守镜像稳定、业务内容可替换原则。
